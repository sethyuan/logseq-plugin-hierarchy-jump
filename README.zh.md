# logseq-plugin-hierarchy-jump

识别当前页面的层级结构并提供快速跳转能力。

## 功能特色

- 支持 namespace 页面。

## Pro 功能特色

还有一个 Pro 版本，它除了这个基础版的全部功能外，还拥有：

- 支持基于标签层级关系的页面。
- 支持标签层级与 namespace 层级的混合层级。
- 可折叠、展开，方便遍历。
- 可快捷键呼出（命令栏也可以）。
- 展开较长的层级时可自动聚焦到当前页面周围，方便快速定位。
